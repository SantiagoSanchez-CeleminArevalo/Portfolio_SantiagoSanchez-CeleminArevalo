Entregables del proyecto

Este repositorio contiene todos los elementos entregados para la evaluación del proyecto:

1. Memoria del proyecto

Incluye:

	Planificación completa

	Planificación de MVP

	Estimación y desglose de costes

	Estadísticas y análisis finales

2. Mockups

	Conjunto de mockups que representan la interfaz propuesta y el flujo principal del sistema.

3. Identidad visual

	Logo del proyecto

Variantes y versiones para distintas aplicaciones

4. Presentación

	Presentación final en formato PDF, utilizada para la exposición del proyecto.

5. Video de demostración

Video explicativo que se muestra al final de la presentación durante el turno de preguntas.
Nota: Los archivos PDF no pueden incrustar correctamente videos en muchos visores, por lo que este archivo se entrega por separado.
