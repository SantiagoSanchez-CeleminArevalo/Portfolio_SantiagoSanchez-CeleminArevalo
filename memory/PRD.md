# Portfolio Santiago Sánchez-Celemín Arévalo

## Problema Original
Transformar un portafolio con temática de placa de circuito en un **escritorio de informático realista** donde cada dispositivo (monitor, teclado, ratón, etc.) representa una sección del portafolio.

## Arquitectura
- **Frontend**: React.js + Tailwind CSS + Framer Motion
- **Backend**: No requerido (portafolio estático)
- **Diseño**: Escritorio de informático con monitor realista
